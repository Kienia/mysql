ТИПЫ ДАННЫХ: 

1. ЧИСЛОВЫЕ ТИПЫ (Целочисленные и с плавающей точкой)

2. СИМВОЛЬНЫЕ ТИПЫ:
Часто используемые: CHAR, VARCHAR, TEXT
                    CHAR - Тип фиксированной длины 
                    VARCHAR - Тип переменной длины (В основном используется) длина от 0 - 65 535
                    TEXT - Для больших данных
                    
* Если известно точное кол-во символов строки использовать СHAR в противном VARCHAR для больших строк TEXT и далее ...

3. ДАТА И ВРЕМЯ
Часто используемые: DATETIME - 0000-00-00 00:00:00 (Полная дата) (Занимает 8 байт) NOW() - Вернет текущее время
                    TIMESTAMP - 00000000000000 (Полная дата в другом способе хранения) (Занимает 4 байта)
                    DATE - 0000-00-00
                    
                    * При записе невалидных значений значения будут преобразованы в 'Ноль'
Другие:
                    TIME - 00:00:00 (Время без даты)
                    YEAR - 0000 (Год)
                    
4.ENUM - Перечисляемый тип
         Использовать если значение одно из ранее известных сначений (Например select > options)
         Записывает не само значение а индекс
                    
* Для всех типов данных если длина значения больше разрешенной данные будут усечены.
